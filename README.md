## :rocket: Como executar
Execute o passo a passo de comandos no seu terminal:
1. ```git clone https://github.com/TheMath123/podcastr.git```
2. ```cd podcastr```
3. ```yarn install```
4. ```yarn build```
5. ```yarn start```
